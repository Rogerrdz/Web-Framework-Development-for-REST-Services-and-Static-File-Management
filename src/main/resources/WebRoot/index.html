Welcome to the Web Framework. Have a great day!

This is a lightweight Java web framework.

Available Routes:

  • GET /pi - Returns PI value
  • GET /euler - Returns Euler value
  • GET /App/pi - Returns PI with label
  • GET /App/helloWorld - Hello World message
  • GET /App/frommethod - Returns Euler from method
  • GET /App/hello - Hello message
  • GET /App/hello?name=Pedro - Hello with name parameter
